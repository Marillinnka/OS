Тесты

Валидные тесты:
-(8*b*-((b*-3)*-8+(-3))) - выражение, содержащее все допустимые переменные и знаки операций
(a+-b*(-a+3)+((a+b)+((a*-b)))+((((a+8)+(a*8)))))- выражение, содержащее большое количество скобок
b+-a - пример с отрицательной переменной
a+(-b) - пример с отрицательной переменной, заключенной в скобках
(b)*a - умножение двух переменных, одна из которых в скобках
-(a+b) - отрицательное выражение
((b)) - переменная в двойных скобках
-(b) - отрицательная переменная в скобках (знак минус за скобками)
(a) - переменная в скобках
(-a) - отрицательная переменная в скобках (знак минус в скобках)
-8 - отрицательная переменная без скобок
-(-a) - отрицание отрицательной переменной
--b - выражение, содержащее несколько минусов перед переменной
b+a*-8+3*-b - выражение, не содержащее скобки

Невалидные тесты:
 - пустая строка
) - закрывающая скобка
( - открывающая скобка
(3 - выражение без закрывающей скобки
8) - выражение без открывающей скобки
() - скобки, не содержащие внутри себя выражения
(()) - двойные скобки, не содержащие внутри себя выражения
(a+b)*(a*3)) - выражение, в котором лишняя закрывающая скобка 
((a*-8)+-((a+b)) - выражение, в котором не хватает закрывающей скобки 
)a - выражение начинается c открывающей скобки
+8+a - выражение начинается с +
*-8+b - выражение начинается с *
3++b - несколько знаков "+" подряд
8**8 - несколько знаков "*" подряд
+**-+ - выражение, которое содержит только знаки операций
883aaab - выражение, содержащее только переменные
6=01/g - выражение, содержащее переменные, не входящие в множество
b8+3 - нет знака операции между переменными
a*(a+) - знак сложения находится в неправильном месте
